# PROJECT

## Project Name

Resto Price Scheduler

---

## Purpose

Menyederhanakan proses perubahan harga WooCommerce.

Saat ini proses perubahan harga dilakukan satu per satu.

Plugin ini akan memungkinkan perubahan harga secara massal menggunakan Excel dan penjadwalan otomatis.

---

## Objectives

- Export Product
- Import Product
- Preview Changes
- Apply Changes
- History
- Scheduler
- Rollback

---

## Long Term Vision

WooCommerce Price Management Suite

## Development Philosophy

Safety First

Semua perubahan harga harus melewati Preview.

Tidak ada perubahan langsung ke database.

Semua perubahan wajib memiliki History.

Scheduler menggunakan engine yang sama dengan Import.
